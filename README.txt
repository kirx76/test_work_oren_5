В файле db.sqlite лежит таблица appuser(id integer,name text).
В папке server -- код сервера flask с поддержкой REST API.
Файл run.py запускает сервер.

Единственный поддерживаемый запрос:

GET /api/users/<int:user_id>

который выдает имя пользователя по его коду.

Пример:

$ curl http://127.0.0.1:5000/api/users/1
"Ivan Ivanov"

Задания:

* Доработать запрос так, чтобы данные по пльзователю выдавались в виде
  json-объекта {"id": <id>, "name": "Ivan Ivanov"}

* Сделать запрос GET /api/users который выдает список пользователей в формате
  json.

* Сделать запрос POST /api/users который вставляет нового пользователя.
